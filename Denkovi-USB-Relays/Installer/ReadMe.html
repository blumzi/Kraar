Abstract.

This is an ASCOM Dome driver for the 8 relays USB switch by Denkovi, at the Kraar Observatory.

Hardware

A board with 8 solid state relays, with USB interface, by Denkovi.

Software

The driver supports a single USB-attached card. The FTDI Id of the card can be set in the driver's Setup window. It defaults to zero.

A set of eight labels can be entered via the Setup window

Relays state persistance

The Setup window has a checkbox for relay value persistence. If it is checked (default):

  • Every five seconds the driver queries the card for the relays state. If a valid value is received, the driver saves it in the ASCOM profile. This ensures that relay state changes performed through other interfaces will be preserved as well.
  • If a valid state is saved in the ASCOM profile, the driver enforces it onto the card at Connect time

If the checkbox is not checked, no relay state persistance will be maintained.

ASCOM Compliance

The driver passes the ASCOM 6.2 Compliance test.
